# lost-and-found
Design and develop a responsive web application that streamlines the process of reporting, searching, and claiming lost items within the college campus. The system should provide a secure and efficient way to reunite students with their lost belongings.

# Purpose
This website shall help people find the lost things in a huge campus as people tend to lose stuff while going around. A solution is a website accessible to all the campus students and faculty to not only be able to find lost items but also report any lost items and provide a contact to anyone who is searching for it.
